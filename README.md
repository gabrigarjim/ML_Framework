# Purpose

This software was created as a merge between tensorflow and R3BRoot. For running the framework you'll need a lot of things installed and compiled.

- Bazel
- Eigen
- Protobuf
- TensorFlow
- CUDA (optional)
- cudNN (optional)
- R3BRoot
- FairRoot
- FairSoft
(Sorry about the long list)
